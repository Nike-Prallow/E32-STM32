# E32-STM32 Example
Simple implementation for a E32-LoRa-Board ReceiveMsg/SendMsg with a STM32 with Arduino (https://github.com/rogerclarkmelbourne/Arduino_STM32).

1. Download Arduino IDE
2. Download and compile E32-TTL-100.ino

| STM32 |           | E32-TTL-100 |
| ------ |:-- |:----:|
| D7     | <-> | M0   |
| D8     | <-> | M1   |
| A0     | <-> | AUX  |
| D10(Rx)| <->    | Tx   |
| D11(Tx)| <->    | Rx   |
